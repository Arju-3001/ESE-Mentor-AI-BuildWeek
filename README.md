# ESE Mentor AI

## 📌 Overview
ESE Mentor AI is an AI-powered web application for Engineering Services Examination (ESE) preparation.

## ✨ Features
- User Registration & Login
- AI Tutor
- PDF Upload & AI Summary
- Ask Questions from PDF
- MCQ Generator
- ESE Mains Answer Generator

## 🛠 Tech Stack
- Python
- Flask
- HTML
- CSS
- SQLite
- Google Gemini AI
- Render

## 🚀 Deployment
Hosted on Render.

## 🤖 AI Usage
This project was developed with assistance from ChatGPT and Codex for:
- Project planning
- Flask development
- Debugging
- AI integration
- UI improvements

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
```
